Yukitter.

ツイートしかできない俺得ツイッタークライアント：Yukitter.
Yukitterはすでに登録されていたので、Yukitter.です。
ピリオドを忘れないように。

How to use

1. すぐに試してみたい方

jarファイルのダウンロードはこちらから -> Yukitter. ver.1.0

2. ソースコードを眺めたい方/自分でカスタマイズしたい方

ソースはこちらから -> GitHubへ飛ぶ

% git clone https://github.com/Yukineko-mew/YukitterTweetOnly.git

実行のためには、Twitter4j-coreとTwitter4j-stream、Twitter4j-asyncが必要です。
各自こちらからダウンロードし、パスを通してください。

実装機能

  • OAuth認証
  • ツイートの送信(Command+Enter/Ctrl+Enterで送信できる)
  • リプライ/お気に入り/リツイートをダイアログと音で通知
  • お気に入り、リツイートダイアログの自動消去(7秒間)
  • アイコンをクリックするとタイムラインが展開する(もう一回クリックするとタイムラインが閉じる)
  • StreamTimeline対応
  • お気に入り/お気に入り解除
  • リツイートができる(注:リツイートの解除はできません)
  • ツイート内の画像をサムネイル表示する
  • サムネイル画像をクリックするとスクリーン中央に元の画像を表示
  • via(クライアント)の表示
  • お気に入り/リツイート数を表示
